# dev_shop

dev_shop is a small demo e‑commerce application built for learning and portfolio purposes.  
It implements core online‑store features such as product listing, product detail pages, a shopping cart, and a basic checkout flow.  
The project is focused on clean architecture, reusable components, and clear separation of concerns to make it easy to extend and refactor.

## Backend tech stack

- Django  
- Django REST Framework (DRF)  
- Redis  
- RabbitMQ  
- PostgreSQL
